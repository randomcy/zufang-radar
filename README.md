# 租房雷达 · Rent Radar

> 帮年轻人租到真正适合自己的房子

48 小时为小红书面试打造的 AI Demo。不做"又一个房源信息聚合"，而是解决三个真实痛点：

1. **我不知道自己想要什么样的房子**
2. **我在几个候选里纠结，选不出来**
3. **房源页不会告诉我住进去真实是什么样**

## 三个核心功能

### 功能一：偏好揭示引擎

用市场研究里的 **Conjoint Analysis（联合分析）** 反推用户的隐性偏好。

- **场景 A**：还不知道想要什么 → 8 道虚构房源选择题 → 输出偏好画像（雷达图 + 权重排序）
- **场景 B**：手里有几个候选 → 10 道两两比较 → 输出候选排序 + 胜负解释

### 功能二：通勤地图筛选

输入公司位置 + 期望通勤时间，反向找出符合条件的地铁站和周边房源。

### 功能三：小区体检报告

聚合小红书真实居住体验，输出小区的优缺点 TOP3、分项评分、引用帖子。这是只有小红书能做的事。

## 技术栈

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- Recharts（图表）
- 高德地图 JS API
- Vercel 部署
